# Rose Bloom UI

Standalone full-screen rose scene that blooms once and then settles into a slow drift.

## Open it

Open [index.html](C:\newprojects\rose-bloom-ui\index.html) directly in a browser.

For a localhost preview, run:

```powershell
node C:\newprojects\rose-bloom-ui\serve.mjs
```

Then visit `http://127.0.0.1:4173`.

## Notes

- No text or controls are rendered on screen.
- The roses bloom once on load, then stay in place with only slight motion.
- The scene is CSS-driven, so there is no runtime interaction cost.
